# Sales_Data_Cleaning_with_PowerQuery

## 📖 Overview
This project demonstrates how to clean and transform raw sales data using **Power Query** in Excel/Power BI.  
It is designed as a practice project to showcase essential data preparation techniques for analytics and reporting.

---

## 📂 Dataset
- **Source:** Raw sales data (CSV/Excel format)
- **Contents:** Transaction details including dates, product IDs, customer info, and sales amounts
- **Issues in raw data:**
  - Missing values
  - Duplicates
  - Inconsistent column names
  - Mixed data types

---

## 🛠 Steps Performed
1. **Importing Data**  
   Load raw sales data into Power Query.

2. **Data Cleaning**  
   - Remove duplicates  
   - Handle missing values  
   - Standardize column names  
   - Correct data types  

3. **Transformations**  
   - Create calculated columns (e.g., profit margin, total sales)  
   - Split/merge columns for better structure  
   - Filter irrelevant records  

4. **Final Output**  
   Export clean, structured data ready for analysis in Excel/Power BI.

---

## 🚀 How to Run
1. Open the project in **Excel** or **Power BI**.  
2. Navigate to **Power Query Editor**.  
3. Review applied steps in the query pane.  
4. Refresh the query to apply transformations to the raw dataset.  

---

## 📊 Example Use Cases

- Generating monthly/quarterly reports
